# JSON VIEWER
----------------

## Installation
You should have server for this to use, because it uses XMLHttpRequest to download <code>JSON</code>

## Basic usage
1. Run server and host this project on it
2. Visit index.html
3. Paste URL to the JSON
4. Click <code>pull data</code> button
5. You can change formatting using checkbox near the button

# Notes
